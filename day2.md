# Git 특강 2일차

#### 원격 저장소 연결

```bash
$ git remote add origin https://github.com/ydo1619/TIL.git
```



#### 원격 저장소 연결 여부 확인

```bash
$ git remote -v

origin  https://github.com/ydo1619/TIL.git (fetch)
origin  https://github.com/ydo1619/TIL.git (push) 
```



#### 추가 파일 업로드

```bash
$ git commit -m "Upload day2"

$ git push -u origin master
```
